# Contributing

Thank you for your interest in GlassEngine.

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a pull request with a clear explanation

## Good first tasks

- Improve the README
- Add documentation for the architecture
- Create a simple issue template
- Add build notes for Visual Studio
- Help design the first widget API
- Improve the initial windowing milestone

## Coding expectations

- Keep code clear and readable
- Prefer small, focused changes
- Add comments where behavior is not obvious
- Document public APIs

## Communication

Use GitHub issues and pull requests for technical discussion. Keep feedback constructive and specific.
