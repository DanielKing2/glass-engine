# Roadmap

## Phase 0 - Foundation

- Create repo structure and documentation
- Set up CMake build system
- Build a simple starter executable

## Phase 1 - Windowing

- Create a transparent window
- Add rounded corners
- Add resize support

## Phase 2 - Rendering

- Render a basic rectangle
- Add text and images
- Measure FPS

## Phase 3 - Capture and compositing

- Capture the desktop
- Composite captured content into the window

## Phase 4 - Blur and glass effects

- Add Gaussian blur
- Add edge lighting
- Add refraction and reflection effects

## Phase 5 - Widget framework

- Introduce a widget base class
- Build a first clock or weather widget

## Phase 6 - Community and open source

- Publish issues and contribution tasks
- Welcome designers, shader developers, and documentation contributors
